-   The max voulme is **1 exabyte**(1,000 TB)
    -   Max file size is **16TB**
    -   Max *4 billion files*
        -   maximum length of a file name is *256* bytes

> \[!tip\]- You can easily upgrade [ext3](/ext3) to ext4 and
> vice versa.
