#alt-name Transmission units were taking a little bit of data and transfering it across the network as a single unit

Examples

  • Ethernet
    • Operates on a frame of data
    • It doesn’t care what’s inside!
  • IP
    • Operates on a packet of data
    • Does not care what’s inside!
      • Ip contains [UDP](/obisdian_ntoes/for later/UDP.md) or TCP or diff protocol

If the data unit contains a header it will contatain: TCP segment or [UDP](/obisdian_ntoes/for later/UDP.md) datagram
