Monitor mode

In monitor mode, a Wi-Fi adapter can passively listen to all the wireless traffic on a specific channel without actually connecting to any network. This mode is useful for tasks such as network analysis, packet sniffing, and intrusion detection.

U can access all the wireless traffic passing by you within the range of your wireless network adapter and antenna (standard is about 300–500 feet)

Activation

Thsi will rename ure [WLAN](/obisdian_ntoes/notes_obsidian/ZPythonref/DjangoFramework/Network+/Network Types/WLAN.md) example: wlan0mon

  • Formula
    • airmon-ng start|stop|restart interface
      • airmon-ng start wlan
