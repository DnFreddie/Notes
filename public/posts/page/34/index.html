systemd_mount

Systemd mount units are used to control the mounting and unmounting of filesystems. #etc They are similar to traditional /etc/fstab entries but are managed by systemd. Mount units are defined in configuration files typically located in /etc/systemd/system or /run/systemd/system. These files have a .mount extension. A unit configuration file whose name ends in ".mount" encodes information about a file system mount point controlled and supervised by systemd. What specifies the device or filesystem to be mounted.
